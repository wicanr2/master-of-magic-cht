# 《魔法大帝》策略筆記與自動建議規則依據

本文整理自多份《Master of Magic》(1994, MicroProse)玩家攻略,目的有二:一是給中文玩家一份
可讀的策略參考;二是作為本專案「自動建議」(Automatic Advice)設定的規則依據——每條建議都對應
攻略裡公認該注意的事,且能由遊戲狀態機械判斷。

> 來源(跨 5+ 份交叉比對):the-spoiler.com 攻略、strangetextsbutcher 的 MoM FAQ、
> Master of Magic Fandom Wiki(Unrest / Town / Food / Gold 機制頁,含具體公式)、StrategyWiki、
> Steam 社群指南(Ten Tips / Recommendations / Help for a noob)、lilura1 的 11-book build 文。
> 數值(每 2 單位 −1 unrest、稅率→unrest 對照、缺食每回合 −50 人、人口上限 = min(25, 食物)、
> Settlers 扣 1000 人口等)均出自原始機制頁,可直接用於程式判斷。

---

## 一、經濟:金幣、食物、法力(最常被攻略強調)

維持單位、建築、結界與召喚都要消耗資源;每回合淨收支為負時,upkeep 視窗數字會變紅。攻略一致強調
「盯緊收支」是最基本也最常見的死因。

- **食物為負**:部隊會**餓死**——每缺 1 食物每回合損失約 50 人口。普通單位每隻吃 1 食物;英雄、奇幻
  單位、被不死化的普通單位食物為 0。解法:增派農民、蓋穀倉(+2 食/+2 上限)與農夫市場(+3/+3)。
- **金幣為負**:國庫(總金幣儲備)見底後,系統會從「最新訓練的單位」開始**強制解散**(不拆建築)。
  解法:加稅、蓋市場/銀行、征服或新建城市、煉金術(法力換金)、賣建築、解僱昂貴部隊/英雄。
  免維護的便宜守軍:Spearmen、Dwarf Swordsmen、貴族英雄。
- **法力為負**:儲備見底後結界與召喚生物會開始消失。解法:調整魔力分配、召喚 (Guardian) Spirit 去
  融合(Meld)節點、蓋神殿系建築、煉金術(金換法力)、銷毀神器釋放法力、解除昂貴召喚。

## 二、城市:叛亂(unrest)與守軍(garrison)

- **叛亂人口(rebels)不繳稅、不產食物/生產,卻照吃 1 食物**——等於純負擔。來源主要是**稅率過高**
  與**征服的異種族城市**(尤其 Klackon / Troll 被別族統治時叛亂多)。
- **鎮壓**:每 **2 個普通單位或英雄**可壓下 1 個叛亂者(奇幻單位不計;Chaos-Channeled 或不死化的
  普通單位例外)。也可蓋宗教建築(Shrine/Temple…)或降稅。
- **守軍**:城市**至少要 1 個單位才能嘗試防禦**;一般建議放 2 個(順便鎮壓叛亂)。
  例外:遇 rampaging monster 來襲且該城是要塞首都時,撤守反而正確(要塞會阻止怪物造巢)。
- 稅率→unrest 對照(每人):0 / ½ / 1 / 1½ / 2 / 2½ / 3 金 = 0 / 10 / 20 / 30 / 45 / 60 / 75%。

## 三、法術研究

- 攻略共識:**法術研究是全程的主要瓶頸**。研究槽空轉(有研究分配卻沒選法術)是最浪費的失誤之一——
  隨時確保有在研究。全部研究完後,把研究分配歸零,轉去法力儲存/施法技能。

## 四、擴張、單位與節點

- **擴張靠征服,不要狂造移民**:多份指南明確主張「少數強城勝過一堆弱城」,別把生產力浪費在大量
  Settlers 上。(注意:人口 < 2000 的城市造 Settlers 會被整個移除;要塞城更是絕對別造。)
- **別讓單位閒置**:純防守被視為致命錯誤;閒置單位該去探索、駐防弱城,或參戰累積經驗。
- **節點(Magic Node)是早期關鍵法力來源**:清空守衛後派 (Guardian) Spirit 去 Meld;有閒置的 Spirit
  就該去 Meld 或探索。lilura1 建議優先用 Guardian Spirit 並對其施 Invulnerability。
- **英雄**:早期關鍵英雄(如 Zaldron 加研究、B'Shan 加金)很快回本;Champion 需聲望(fame)≥ 40。

## 五、進階城市附魔

研究到後可在主要生產城施放城市附魔以解鎖/強化:Tree of Knowledge(解鎖頂級單位建築)、
Inspirations(+生產)、Stream of Life(配合滿稅抑制叛亂)。

## 六、自動建議規則(本專案實作)

把上面可機械判斷的核心建議做進「自動建議」設定(預設關;進階玩家會關掉)。開啟時,於人類回合開始
檢查狀態,符合就以卷軸提示。已實作的規則:

| 觸發條件(遊戲狀態) | 建議 | 依據 |
|---|---|---|
| 帝國金幣每回合 < 0 | 提高稅率、蓋市場,或解僱昂貴部隊 | 一、金幣 |
| 帝國食物每回合 < 0 | 部隊將餓死,增派農民 | 一、食物 |
| 帝國法力每回合 < 0 | 調整法力分配、融合節點,或蓋宗教建築 | 一、法力 |
| 沒有研究中的法術 | 到法術書選擇研究目標 | 三、研究 |
| 某城市有叛亂人口 | 增加守軍、蓋宗教建築,或降稅 | 二、叛亂 |
| 某城市無守軍 | 無法防禦,放置守軍 | 二、守軍 |

> 規則由攻略歸納而成、每條都有上述依據,但**非逐字重現原版彈窗文字**(原版確切文字未公開於手冊)。
> 後續可擴充的規則(已調查、見下)尚未全部實作:未 Meld 的鄰近節點、閒置單位/Spirit、單城金幣赤字、
> 成長建築缺漏、已達人口上限、可雇英雄但缺金、建造前置/地形不符、可施放但未施的城市附魔、生產溢出
> 轉 Trade Goods 等。實作時兩點注意:**不要建議「狂造移民」**(與社群共識相反),**「無守軍」提示遇
> rampaging monster 來襲的要塞城要例外**。
